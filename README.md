# Scholarship Management System

C++ console application to automate student scholarship evaluation and distribution.

## Features
- Automatic student registration and record storage
- Merit-based and need-based scholarship eligibility
- Multi-tier scholarship calculation (25%, 50%, 75%, 100%)
- Department-wise distribution and reporting
- Menu-driven CLI interface

## Technologies Used
- C++ OOP, Enums, Vectors
- Input validation and error handling

## How to Run
1. Open terminal
2. Compile: `g++ main.cpp -o ScholarshipSystem`
3. Run: `./ScholarshipSystem`
